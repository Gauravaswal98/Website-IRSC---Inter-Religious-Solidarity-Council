### Digital Platform for the Inter-Religious Solidarity Council (IRSC)

IRSC is a premium, high-impact web platform developed for the **:contentReference[oaicite:0]{index=0} (IRSC)**.

## üõ† Technology Stack

The platform is built with a strong focus on performance, accessibility, and modern UI design:

- **HTML5 & CSS3**  
  Semantic structure for accessibility, SEO, and maintainability.

- **Tailwind CSS**  
  Utility-first framework for a premium UI with fluid layouts, modern typography, and a refined color palette.

- **JavaScript (ES6+)**  
  Powers interactive components, dynamic content rendering, and volunteer dashboards.

- **Lucide React / Font Awesome**  
  Clean and consistent iconography for an intuitive user experience.

- **Google Fonts (Noto Sans)**  
  Global character support ensuring excellent readability across languages.

## üìÑ Platform Structure

IRSC is organized into modular sections for a seamless user journey:

### 1. Home / Landing Page
- Hero section with a strong call-to-action for global solidarity  
- Real-time impact metrics (peace grants, lives touched)  
- Featured highlights from the 2026 Resilience Project cycle  

### 2. Resilience Projects
- Dedicated archive of social impact and peacebuilding stories  
- Visual galleries and detailed interfaith cooperation case studies  

### 3. Volunteer Portal
- Skill development resources for peacebuilding and conflict resolution  
- Automated certification and volunteer hour tracking  
- Personal impact dashboard for active contributors  

### 4. Global Advocacy & News
- Updates from international inter-religious summits  
- Press room with official announcements and media resources
